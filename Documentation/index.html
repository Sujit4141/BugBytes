01 / Overview

What is MineWatch?

MineWatch is a real-time fatigue detection system for coal mine workers. It uses webcam-based computer vision to track eye aspect ratio, blink rate, and head pose — combined with environmental sensor data — to classify fatigue levels into three categories.

● NORMAL

Alert and focused. Safe to continue operations without restrictions.

◆ MODERATE

Fatigue building. Schedule immediate 15-minute rest break now.

▲ HIGH RISK

Dangerous fatigue detected. STOP — report to supervisor immediately.

04 / Features

Key Features

MineWatch combines real-time computer vision with environmental sensor fusion to give a complete picture of operator fatigue.

👁

EAR Detection

Eye Aspect Ratio calculated from 6 MediaPipe landmarks per eye. EAR below 0.21 = eyes closed. Uses left + right average.

🔁

Blink Rate

Counts blinks per minute using consecutive EAR drop detection. Requires 2+ consecutive closed frames to register a blink.

📐

Head Pose

Pitch, yaw, roll extracted from 4×4 facial transformation matrix. Head tilt = average absolute roll across the session.

⚡

Reaction Test

4-round colour-flash test. Ratings: Excellent <200ms, Good <300ms, Moderate <400ms, Slow/Fatigued ≥400ms.

🌡

Sensor Fusion

Combines vision data with heart rate, shift hours, pit temperature, and gas level for comprehensive fatigue scoring.

🤖

ML Classification

Decision Tree Classifier trained on fatigue dataset. 7 input features, 3 output classes. Serialized with joblib.

05 / Setup

Local Development

Clone the repo, install dependencies, and run both services locally. Backend needs Python 3.9+, frontend needs Node 18+.

Backend
# Go to backend folder
cd backend/
 
# Install dependencies
pip install -r requirements.txt
 
# Run dev server
python app.py
 
# Runs at http://localhost:5000
Frontend
# Install node packages
npm install
 
# Start dev server
npm run dev
 
# Production build
npm run build
 
# Deploy to Vercel via git push
